When would you want to use a remote repository rather than keeping all your work local?

Remote repository will keep work accessible when not on local computer.  It will also help with collaboration.  

Why might you want to always pull changes manually rather than having Git automatically stay up-to-date with your remote repository?

Automatic updates could interfere with local work.

Describe the differences between forks, clones, and branches.  When would you use one instead of another?

Branches are on a single repository, clones are a copy of existing repository (can be between one or two repositories), and forks are a specific clones from one GitHub repository to another.
